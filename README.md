# 🚗 Vigil Drive Sense: Driver Drowsiness & Emotion Detection

A Deep Learning project that detects driver drowsiness and facial emotions using **Transfer Learning with ResNet18**. The objective is to improve road safety by identifying signs of fatigue and emotional states from facial images.

---

## 📖 Overview

Driver fatigue is one of the leading causes of road accidents. This project explores the use of Computer Vision and Deep Learning to automatically classify driver states such as eye closure, yawning, and facial emotions.

The model utilizes a pre-trained **ResNet18** network and applies transfer learning to improve classification performance while reducing training time.

This project was developed as part of the **Deep Learning** coursework at **VIT Chennai**.

This project references publicly available datasets and resources.

### Emotion Recognition Dataset

https://www.kaggle.com/datasets/ananthu017/emotion-detection-fer

The FER2013 dataset contains over 35,000 grayscale facial images categorized into seven facial emotions.

### Drowsiness Detection Reference
